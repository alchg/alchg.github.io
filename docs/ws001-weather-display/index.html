    • 乾電池式お天気ディスプレイ


    ディスプレイに電子ペーパーを採用し、画面更新時以外はスリープモードに入ることによって長期間動作します。
    またこのガジェットはオープンソースとして公開しています。

    主な機能

    • Bluetooth BLEを使用した初期設定
    • 気象庁の提供JSONデータによる全国天気予報表示
    • 一日４度の天気情報更新
    • 単３乾電池３本USB電源（有線USB電源も可能）
    • 連続動作３ヶ月以上（ニッケル水素充電池1950mAhによる）
